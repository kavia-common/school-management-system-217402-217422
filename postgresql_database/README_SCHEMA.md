## Apply schema + seed

1) Start the database:

```bash
cd postgresql_database
./startup.sh
```

2) Apply schema + seed:

```bash
cd postgresql_database
chmod +x apply_schema.sh
./apply_schema.sh
```

`apply_schema.sh` reads the connection string from `db_connection.txt` (generated by `startup.sh`) and runs `schema_and_seed.sql`.

## Notes

- `password_hash` values are demo placeholders (not secure). Real backend code should store strong password hashes (bcrypt/argon2).
- The script drops and recreates tables (safe for local demos).
- Seed academic year used: `2025-2026`.
